# ディレクトリ構成
ディレクトリ構成は下記の通りとする。
```
/
├─ .github
├─ .vscode
├─ docs
└─ src
   ├─ html
   ├─ style
   └─ js
      └─ modules

.github：GitHubに関するファイルを格納する
.vscode：Visual Studio Codeに関するファイルを格納する
docs：ドキュメントに関するファイルを格納する
src：テトリスに関するソースコードを格納する
html：HTMLファイルを格納する
style：CSSファイルを格納する
js：JavaScriptファイルを格納する
modules：JavaScript内で使用するモジュールを格納する
```
